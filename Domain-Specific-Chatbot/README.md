# Domain Specific Chatbot

## Description
An AI-powered career guidance chatbot that uses BERT for intent classification, entity extraction, and Google Gemini to generate intelligent responses.

## Features
- Career guidance
- Skill recommendations
- Certification suggestions
- Interview preparation
- Resume guidance
- AI-powered chatbot

## Technologies Used
- Python
- Google Gen AI
- Transformers (BERT)
- Gradio
