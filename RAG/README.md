# RAG PDF Chatbot

## Description
An AI-powered Retrieval-Augmented Generation (RAG) chatbot that allows users to upload PDF documents, index their contents, and ask questions based on the uploaded document.

## Features
- PDF Upload
- Text Extraction
- Document Indexing
- Semantic Search
- AI Question Answering

## Technologies Used
- Python
- Google Generative AI
- Gradio
- ChromaDB
- Sentence Transformers
- PyPDF
