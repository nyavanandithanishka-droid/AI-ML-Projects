# AI Weather Agent

## Description
An AI-powered weather assistant that extracts a city name from a user's query, retrieves live weather data using the Open-Meteo API, and generates a friendly weather report using Google Gemini.

## Features
- City name extraction
- Live weather information
- AI-generated weather explanation
- User-friendly interface

## Technologies Used
- Python
- Google Gen AI
- Gradio
- Open-Meteo API
- Requests
